A cross-platform GUI wrapper for [<u>*yt-dlp*</u>](https://github.com/yt-dlp/yt-dlp) for
downloading YouTube videos. Users can see their download history, download progress, and choose
from various colored themes.
<br><br>
Users can install the GUI with the provided installer on Windows or macOS or build it from source.
